# Screenshot Slots

Place real product screenshots in this folder with the exact filenames below.
The homepage automatically shows the image when the file exists, and shows a styled placeholder while it is missing.

- `main-dashboard.png`: main calendar dashboard with panels
- `widget-manager.png`: overlay widget manager or widget examples
- `focus-mode.png`: focus mode or Pomodoro screen
- `calendar-sync.png`: calendar sync or settings screen

Recommended ratio: 16:10 or 16:9.
Recommended width: 1400px or larger.

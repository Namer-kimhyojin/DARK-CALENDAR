Core Message

필요한 것은 더 가까이. 나머지는 더 조용히.

일정, 루틴, 실행 항목, 집중 도구를 한 화면에서 다룹니다. 복잡함은 줄이고 흐름은 이어갑니다.

01

자유로운 패널

도킹, 탭, 플로팅으로 화면을 조립합니다.

02

루틴과 실행 분리

반복할 일과 바로 할 일을 따로 봅니다.

03

집중 모드

선택한 작업을 타이머와 함께 밀어줍니다.

04

Google 연동

외부 일정과 데스크톱 흐름을 연결합니다.

05

레이아웃 프리셋

상황에 맞는 화면으로 빠르게 전환합니다.

06

잠금과 자리 비움

정리한 화면을 안전하게 유지합니다.
